## Publier gratuitement (2 options simples)

### Option A — GitHub Pages (recommandé)
1. Crée un compte GitHub (si besoin) et un dépôt (repo) nommé par ex. `portfolio`.
2. Téléverse tous les fichiers de ce dossier dans le repo (à la racine).
3. Va dans **Settings → Pages**.
4. Source: **Deploy from a branch**, Branch: `main` et dossier `/root`.
5. Enregistre. GitHub te donne un lien du type `https://ton-compte.github.io/portfolio`.

### Option B — Netlify (très simple)
1. Va sur Netlify et connecte-toi.
2. Glisse-dépose le dossier (ou le ZIP) dans la zone “Deploy”.
3. Netlify te donne un lien immédiatement. Tu peux ensuite connecter un domaine.
